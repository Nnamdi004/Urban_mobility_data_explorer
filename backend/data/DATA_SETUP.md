# Data Setup Instructions

This project requires NYC Taxi Trip data that is too large to store in GitHub.

## Required Files

Place the following files in `backend/data/raw/`:

### 1. Yellow Taxi Trip Data (January 2019)
- **File**: `yellow_tripdata_2019-01.csv`
- **Source**: [NYC TLC Trip Record Data](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page)
- **Direct Download**: https://d37ci6vzurychx.cloudfront.net/trip-data/yellow_tripdata_2019-01.parquet
- **Note**: Download as `.parquet` and convert to CSV, or find CSV version
- **Size**: ~700MB

### 2. Taxi Zone Lookup (Already Included)
- **File**: `taxi_zone_lookup.csv`
- **Status**: ✅ Already in repository (small file)

### 3. Taxi Zones GeoJSON (Optional)
- **File**: `taxi_zones.zip`
- **Source**: [NYC Taxi Zones](https://data.cityofnewyork.us/Transportation/NYC-Taxi-Zones/d3c5-ddgc)
- **Status**: ✅ Already in repository

## Quick Setup

1. Download the trip data file
2. Place it in `backend/data/raw/`
3. Run the cleaning script:
   ```bash
   cd backend/script
   python clean_data.py
   ```

## File Structure After Setup

```
backend/data/
├── raw/
│   ├── yellow_tripdata_2019-01.csv  (you download this)
│   ├── taxi_zone_lookup.csv         (already included)
│   └── taxi_zones.zip               (already included)
├── processed/
│   └── cleaned_trips.csv            (generated by script)
└── logs/
    └── transparency_log.txt         (generated by script)
```

## Why Not Include in GitHub?

The trip data file exceeds GitHub's 100MB file size limit. This is standard practice for data science projects.
